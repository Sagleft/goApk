# goApk
Golang APK Downloader

## INSTALLATION
* Install [Golang](https://golang.org/dl/)
* `$ go get "github.com/cavaliercoder/grab"`
* `$ go get ""github.com/buger/goterm"`

## USAGE
* `$ go build` to compile or `$ go run main.go`
